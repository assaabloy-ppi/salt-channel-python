**Same as 'salt-aa' in Java version**

A Python API to TweetNaCl.  
The following subset of the 25 functions in [TWEET-1] are supported:     

* crypto_sign_keypair.    
* crypto_sign.
* crypto_sign_open.   
* crypto_box_keypair. 
* crypto_box_beforenm.    
* crypto_box_afternm. 
* crypto_box_open_afternm.    
* crypto_hash.
